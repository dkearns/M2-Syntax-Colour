# M2-Syntax-Colour
Syntax Descriptions for Colouring/Highlighting Modula-2

### Supported Modula-2 Dialects
* Classic Modula-2 (aka PIM)
* ISO Modula-2 (ISO 10514)
* Modula-2 R10

### Supported Frameworks
* GeSHi (PIM only)
* Highlight (PIM only)
* Pygments

### Supported Editors/IDEs
* BBEdit/TextWrangler
* Kate (Part of KDE)
* Vim/vi
